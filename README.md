# 簡介

使用 Vue3所打造的訂房網站，以Vite建立開發環境，使用Firebase資料庫，Vue Router處理路由，Element Plus UI框架和實現 SPA 單頁應用。

# 技術
- Vue.js
- Vite
- RWD
- JavaScript
- Vue Router
- Element Plus
- Firebase Firestore Database 及 Authentication
- Day.js
- AOS.js

# 架構
- 首頁:廣告輪播圖
- 導覽頁:一般資訊介紹
- 訂房頁:快速訂房(篩選房型)，一般訂房(顯示所有剩餘空房)
- 會員頁:登入，註冊，修改密碼，修改Email，訂單列表
